# 连接服务器

## 连接无线网络

上机前请先连接现场无线网络：

- 无线网络名称：`SMART-Guest`

如无法连接无线网络，请联系现场工作人员或助教。

## 打开个人上机入口

每位学员都有一个个人上机入口。入口统一使用英文账号，不使用中文姓名作为网址。

入口格式如下：

```text
课程网站地址/英文账号.html
```

其中，英文账号对应课程分配的 Linux 文件夹名。请以组织方发送给你的个人入口或英文账号为准。

## 进入个人上机环境

打开个人上机入口后，页面会自动跳转到对应的个人上机环境。如果没有自动跳转，请点击页面中的“进入个人上机环境”链接。

如果页面无法打开，请先确认：

- 已连接 `SMART-Guest` 无线网络。
- 使用的是英文账号入口，而不是中文姓名入口。
- 网址末尾包含 `.html`。

仍无法进入时，请联系现场助教处理。
