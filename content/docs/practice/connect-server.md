# 连接服务器

## 连接无线网络

上机前请先连接现场无线网络：

- 无线网络名称：`SMART-Guest`

如无法连接无线网络，请联系现场工作人员或助教。

## 打开个人上机入口

每位学员都有一个个人上机入口。入口使用英文账号，英文账号对应课程分配的 Linux 文件夹名。

入口格式如下：

```text
http://192.168.61.30/英文账号.html
```

英文账号采用姓名拼音小写，并用下划线连接。两字姓名写作“姓_名”；三字姓名写作“姓_名字的第一个字_名字的第二个字”。例如：

- 林丹：`lin_dan.html`
- 李宗伟：`li_zong_wei.html`
- 张三丰：`zhang_san_feng.html`

请以组织方发送给你的个人入口或英文账号为准。

## 进入个人上机环境

打开个人上机入口后，页面会自动跳转到对应的个人上机环境。页面停留时，请点击页面中的“进入个人上机环境”链接。

如果页面无法打开，请先确认：

- 已连接 `SMART-Guest` 无线网络。
- 英文账号拼写与组织方发送的信息一致。
- 网址末尾包含 `.html`。

仍无法进入时，请联系现场助教处理。

## 在课程服务器之外复现环境

本次上机使用已经配置好的课程服务器，现场环境和部分数据由组织方提前准备。希望在个人电脑、所在学校服务器或所在单位计算平台上继续复现练习的学员，可参考[课程软件环境配置脚本](https://github.com/StatFunGen/pixi-setup)。

课程数据压缩包将上传到基因库服务器。学员可在下载后结合自己的计算环境继续练习，也可以按照所在单位的登录方式和计算资源管理方式完成后续配置。
