# 自建上机环境

本次上机使用已经配置好的课程服务器，现场环境和部分数据由组织方提前准备。课程服务器适合课堂练习、助教支持和统一排错。

希望在个人电脑、所在学校服务器或所在单位计算平台上继续复现课程流程的学员，可参考以下脚本配置课程所需软件环境：

- [课程软件环境配置脚本](https://github.com/StatFunGen/pixi-setup)

在学校或单位服务器上配置时，可结合本单位的登录方式、账号管理方式和计算资源管理方式执行。课程数据压缩包将上传到基因库服务器；学员可下载后放入自己的计算环境，用于课后复现和继续练习。

上机练习的重点是帮助理解数据结构、模型输入输出、分析流程和结果解释。掌握核心原理后，同一套流程可以迁移到新的数据、服务器和研究问题中继续使用。
