# 课程概览

第四届大鹏湾统计功能基因组学暑期课程以“从全基因组关联分析（GWAS）到多基因预测与多组学整合”为主题，地点为深圳大鹏湾国家基因库，时间为2026年6月22日至26日。

课程以中文讲授，教材资料为英文版本。课程面向医学科研人员、研究生、临床工作者，以及动植物遗传、数量遗传与流行病学研究者；学员需具备遗传学、基因组学与统计学基础。
