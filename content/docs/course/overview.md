# 课程简介

统计遗传学以数学语言刻画遗传变异的效应；功能基因组学以高通量测序描绘变异的分子后果。统计功能基因组学位于两者交汇处：从全基因组关联分析（GWAS）信号出发，穿过数量性状位点（QTL）与调控网络，抵达疾病机制，既需要严谨的统计推断，也依赖对细胞类型、组织背景与族群差异的生物学理解。

过去十年间，贝叶斯精细定位、多研究共定位、转录组关联分析（TWAS）、跨族群多基因风险评分（PRS）、功能注释与调控预测正在重塑这一链条的每一环，也带来前所未有的方法学挑战。

第四届大鹏湾统计功能基因组学暑期课程沿这一链条展开，课程历时五天，地点为深圳大鹏湾国家基因库。课程以中文讲授，教材资料为英文版本。

## 理论课程学习目的

课程围绕一个核心问题展开：如何整合多组学数据，解析复杂疾病和复杂性状的遗传机制？

本课程把复杂疾病和复杂性状的分子遗传机制作为主线，关注如何利用分子层面的数据帮助解释复杂遗传信号。上机实践用于帮助学员理解数据结构、分析流程和结果解释；理论课程核心关注方法学原理、统计模型的广义含义，以及这些模型为什么能服务于分子机制解析。

课程强调基础理论与学科前沿结合。周一的统计推断和贝叶斯模型部分强调更通用的建模思想，可延伸到生物信息学等多个场景；周二至周五聚焦这些方法在统计功能基因组学中的具体应用。希望学员通过统计遗传学案例掌握经典模型和建模思维，在新的方法、数据类型和研究问题中举一反三。

## 适合人群

- 医学科研人员
- 研究生
- 临床工作者
- 动植物遗传、数量遗传与流行病学研究者

学员需具备遗传学、基因组学与统计学基础。
