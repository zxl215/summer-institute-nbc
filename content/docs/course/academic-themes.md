# 学术内容

课程围绕统计功能基因组学的完整研究链条展开，从全基因组关联分析（GWAS）信号识别，到分子调控解释，再到疾病机制、预测模型与跨族群应用。

## 五大学术主题

### 全基因组关联分析（GWAS）方法论与贝叶斯精细定位

课程讨论全基因组关联分析（GWAS）研究设计、关联统计量、线性混合模型、群体结构控制、贝叶斯精细定位与多研究共定位，重点理解单性状、多性状和多研究整合的精细定位思想。

### 多组学数量性状位点（QTL）与细胞类型特异性调控

课程覆盖多组学数量性状位点（QTL）的测量、质控与解读，包括表达、剪接、蛋白质组、多组织和细胞类型特异性分子调控信号，以及这些信号如何帮助解释全基因组关联分析（GWAS）结果。

### 转录组关联分析（TWAS）和蛋白质组关联分析（PWAS）、共定位与孟德尔随机化

课程讨论转录组关联分析（TWAS）和蛋白质组关联分析（PWAS）模型、汇总统计量层面的关联分析、条件精细定位、共定位分析、孟德尔随机化，以及这些方法在阿尔茨海默病等复杂疾病研究中的应用。

### 多基因风险评分（PRS）与跨族群迁移

课程讨论多基因风险评分（PRS）构建、模型选择、预测性能评估、跨族群可迁移性、多族群联合建模，以及族群差异对方法学设计与临床转化的影响。

### 反式调控、基因网络与全基因模型

课程讨论蛋白质组、蛋白质数量性状位点（pQTL）、反式表达调控、基因网络与全基因模型，帮助学员理解复杂性状遗传结构中核心基因与外围基因的关系。
