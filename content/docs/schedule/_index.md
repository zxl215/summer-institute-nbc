# 课程安排

课程时间为2026年6月22日至26日，共五天。上午和下午为主题讲座，下午后段根据日期安排讲师答疑、专题讨论、上机实践或反馈意见收集。下午讲座可根据讲师安排在15:30至16:00之间结束；周二至周四答疑随讲座结束时间和现场问题量灵活安排，最晚17:00结束。答疑环节首先开放学员自由提问，随后围绕每天设置的机器学习（ML）与人工智能（AI）专题展开；周五安排全体讲师圆桌讨论。

## 五天总览

| 日期 | 主讲 | 主题 |
| --- | --- | --- |
| 6月22日 星期一 | 董睿 | 频率论统计推断、贝叶斯统计推断、全基因组关联分析（GWAS）基础导论 |
| 6月23日 星期二 | 王告、刘毓文 | 全基因组关联分析（GWAS）方法论、精细定位与共定位新进展、高通量功能基因组实验 |

| 6月26日 星期五 | 刘轩尧 | 多基因风险评分（PRS）、跨族群遗传发现、蛋白质组与全基因模型 |
